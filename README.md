# 🩺 AI Scribe: Clinical Assistant Powered by Gemini Flash 2.0

AI Scribe is an intelligent, LLM-powered tool designed to streamline clinical documentation in real-time. It enables healthcare professionals to effortlessly convert speech, text, and images into structured and insightful medical records.

---

## 🚀 Key Features

### 🎙 Speech to Text
- Real-time transcription of spoken clinical notes.
- Supports multiple accents and medical terminology.

### 🧠 Multiple Summary Types
- *SOAP Format*: Structured medical summaries (Subjective, Objective, Assessment, Plan).
- *Narrative*: Natural language, paragraph-style summaries.
- *Bullet Points*: Clear, concise summaries in list format.

### 🌐 Language Translation
- Translate clinical documentation into multiple languages.
- Enables support for multilingual healthcare environments.

### 🖼 OCR – File & Image Reading
- Extracts medical text from scanned images and uploaded documents.
- Supports JPG, PNG, and PDF formats.
- Useful for digitizing handwritten notes or printed records.

### ⚡ Powered by Gemini Flash 2.0 API
- Fast and accurate LLM-based understanding and generation.
- Enhanced support for complex medical language and summarization tasks.

---

## 🧩 Tech Stack

- *Python 3.12+*
- *Streamlit* (Frontend UI)
- *spaCy* (Text analysis)
- *Gemini Flash 2.0 API* (LLM backend)
- *SpeechRecognition / Whisper API* (for audio processing)
- *Tesseract OCR* (for image-to-text)
